# Social-Media-Content-Automation-Make.com
Add a row to a Google Sheet. Walk away. LinkedIn, Facebook, and Telegram handle themselves.

The problem it solves:

Posting the same content across multiple platforms manually is repetitive and easy to forget. 

This workflow watches a Google Sheet and does the whole thing automatically — summarize, format, post.

How it works:

Google Sheets — trigger fires when a new row is added to the content calendar

Text Parser / AI Summarizer — condenses or reformats the content per platform requirements

Router — splits the flow based on which platforms are flagged in the sheet

LinkedIn / Facebook / Telegram modules — each gets a platform-appropriate version of the post
